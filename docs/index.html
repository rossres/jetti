─── thirty-second tour ───

The whole loop, in three steps.

One command in your agent. One link to your reviewer. One handoff back to your codebase.

no shortcuts skipped ↘
1 install

Install in your agent.

install jetti is a real prompt your AI dev tool understands. We publish a public install contract; your agent reads it and patches your project.

▸ install jetti_
2 review

Reviewers edit live.

Your client opens a link and edits the words and photos they mean — on the actual site, in their browser. No comments to translate. No screenshots to interpret.

click → type → done
3 apply

Hand back to your agent.

npx jetti apply stages the changes on a branch and opens a draft PR. Deterministic patches for text. Photo files saved to disk. Comments left for you to read.
